欢迎使用APKIDE少月版3.3.2版本提供的oat文件反编插件―Oatract1.0。

特别感谢：越狱

工具介绍：该工具是一个辅助工具，用于弥补逆向工具在处理安卓5.0以上的oat文件存在的不足。目前，这样的工具在安卓版较多，但电脑端的明显不足。

使用方法：
1.先把待逆向的oat文件命名为1.oat，替换目录中的1.oat文件
2.按数字次序点击bat文件，先将oat转换为dex，再对dex文件进行去优化和检测是否完全正确。

注意：
对于dex文件去优化与检测，如果没有输出任何文件或一闪而过，也可能是因为dex文件本身正确，也可能是依然存在优化，部分类存在错误，可以找到错误的类，然后反编译为smali，进行静态修复。但这并不说明全部类都存在错误，如可能部分支持库中的类由于使用混淆发生错误。当然，如果仅仅需要提取dex进行分析，那么只进行oat文件转换为dex文件就足够。

如果点击bat文件无效，我们可以手工进入目录，然后在控制台输入以下2条命令：
java -jar oatract.jar boot 1.oat
java -jar oatract.jar 1.oat odex
