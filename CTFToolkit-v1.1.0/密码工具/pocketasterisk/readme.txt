Description
===========

PocketAsterisk is a small utility for Pocket PC devices that reveals that
passwords stored behind the asterisks ('***') in standard password
text-boxes.
You can use it to recover the passwords of POP3/IMAP accounts stored by
the 'Messaging' application of Pocket PC or to recover any other password
that is displayed as asterisks.



System Requirements
===================

This utility should work on any Pocket PC device with ARM processor.



Installing and Using PocketAsterisk
===================================

In order to install and use PocketAsterisk utility, follow the
instructions below:
1. Connect the Pocket PC device to your desktop computer.
2. In ActiveSync program, choose 'Explore', and go to '\Windows\Start
   Menu\Programs'
3. Drag the executable file (PocketAsterisk.exe) from a folder on your
   desktop computer into the '\Windows\Start Menu\Programs' folder. If
   you get any dialog-box, click 'OK' to continue.



4. In the device, open the program the has the asterisk text-box. In
   our example, it's a POP3 account of the 'Messaging' program:



5. Click the start button, and choose programs. you should find here a
   new icon of the PocketAsterisk utility.



6. Click the icon of PocketAsterisk utility, and then close the
   'Programs' window.
7. After closing the 'Program' window, you should see the real
   password instead of asterisks:



8. After you follow the above steps, the icon of PocketAsterisk may
   also be added into the start menu, so we'll be able to recover more
   asterisk passwords without opening the 'Programs' window again.




If you want to remove PocketAsterisk from your device, simply delete it
from '\Windows\Start Menu\Programs'.
